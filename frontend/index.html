Quick start

Run your first agent attack campaign.

Select a target mode, copy the snippet, and run SAGSBench locally. The buttons below copy working commands.

install-and-scan.sh
pip install sagsbench

sagsbench scan \
  --target http://localhost:8000/chat \
  --profile enterprise \
  --report html,json,csv,sarif
http-target.py
from sagsbench import Campaign, HTTPAgentTarget
from sagsbench.attacks import BaselineAttackSuite
from sagsbench.judges import RegexJudge, PolicyJudge

campaign = Campaign(
    target=HTTPAgentTarget(
        name="customer-support-agent",
        endpoint="http://localhost:8000/chat",
        input_key="message",
        output_key="response",
    ),
    suites=[BaselineAttackSuite.from_builtin("enterprise")],
    judges=[RegexJudge(), PolicyJudge.from_builtin("sags_baseline")],
)

campaign.run().to_html("reports/sagsbench-report.html")
evaluate-defenses.py
from sagsbench import evaluate_defenses, make_defense_stack, HTTPAgentTarget
from sagsbench.attacks import BaselineAttackSuite
from sagsbench.judges import RegexJudge, PolicyJudge

result = evaluate_defenses(
    target=HTTPAgentTarget(name="agent", endpoint="http://localhost:8000/chat"),
    attack_suite=BaselineAttackSuite.from_builtin("enterprise"),
    benign_suite=BaselineAttackSuite.from_builtin("benign"),
    variants=[
        ("no-defense", make_defense_stack("no-defense")),
        ("tool-filter", make_defense_stack("tool-filter")),
        ("combined", make_defense_stack("policy-engine+approval-gate+tool-filter")),
    ],
    judges=[RegexJudge(), PolicyJudge.from_builtin("sags_baseline")],
)
print("Recommended stack:", result.best_variant)
result.to_html("reports/sagsbench-defenses.html")
openai-compatible.py
from sagsbench import Campaign
from sagsbench.adapters import OpenAICompatibleTarget
from sagsbench.attacks import BaselineAttackSuite
from sagsbench.judges import RegexJudge, PolicyJudge

campaign = Campaign(
    target=OpenAICompatibleTarget(
        name="governed-agent",
        base_url="https://api.openai.com/v1",
        model="gpt-4o-mini",
    ),
    suites=[BaselineAttackSuite.from_builtin("enterprise")],
    judges=[RegexJudge(), PolicyJudge.from_builtin("sags_baseline")],
)

campaign.run().to_sarif("reports/sagsbench.sarif")

New · Defense Evaluation Mode

Prove which defenses actually reduce risk.

Run the same attack and governance suites against a baseline and one or more defense variants. SAGSBench replays identical cases, then compares attack success, governance failures, task completion, false positives, latency, and audit completeness — and recommends a stack.

New · Public Leaderboard

Rank agent stacks, not just models.

Enterprise agent risk depends on the whole architecture — model + framework + tools + memory + approval flow + policy layer + monitoring. The SAGSBench leaderboard ranks stacks by track with verified, reproducible, and community-checked submissions.

01

5 use-case tracks

Customer support, finance/procurement, HR, DevOps/SRE, and multi-agent governance — each with its own risks.

02

Verification levels

Self-reported, reproducible, community-verified, maintainer-verified, and private enterprise boards.

03

Anti-gaming by design

Hidden splits, versioned suites, audit-evidence requirements, and a task-utility floor so agents can’t win by refusing everything.
