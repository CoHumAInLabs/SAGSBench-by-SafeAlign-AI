# Contributing to SAGSBench

Thank you for helping improve open-source red teaming for agentic AI governance.

## Good first contributions

- Add safe synthetic red-team cases
- Add deterministic judges
- Add target adapters
- Improve report exports
- Improve policy mappings
- Add examples for popular agent frameworks

## Responsible-use rules

Do not contribute payloads intended to break into third-party systems, steal credentials, evade detection, or enable unauthorized testing. SAGSBench is for defensive, authorized assessment.
